This repository contains a static website managed using Eleventy (https://www.11ty.dev/). The website provides information about the Cape Ann Amateur Radio Assocation.

- Source content is in the `content/` directory.
- The main css stylesheet is `content/css/style.css`.
- Create the rendered content by running `npm run build`.
- Rendered content can be found in the `_site/` directory.

## Standards

- No inline css.
- Reuse code and templates when possible.
- Cache API operations to avoid quotas.
